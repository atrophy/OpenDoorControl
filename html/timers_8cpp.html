DESCRIPTION

The timers are the core of OpenDoorControl, in order to remove blocking code and allow for asynchronous operation most of the functionality is tied to one or more timers.

void serviceTimers ( )

Poll the timers and update or refresh as needed

Checks the declared timers for expiries and runs their required expire function. Dependant on the repeat flag may either stop or reset the timer.
